YMV Smart Filter - Filter posts/custom post types by custom taxonomy/category without page reload and with pagination too. It has different filter and post layouts.

== Description ==

YMC Smart Filter - allows to solve a variety of tasks for displaying posts on site pages.

== Installation ==

1. Activate Plugin or upload the entire 'ymc-smart-filters' folder to the '/wp-content/plugins/' directory.
2. Add new YMC Smart Filter
3. Copy YMC Smart Filter shortcode and paste to any page or post
4. Set setting for each post.
